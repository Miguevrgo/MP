{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}\item[{int}]{argc,  }\item[{char $\ast$}]{argv\mbox{[}$\,$\mbox{]} }\end{DoxyParamCaption})}

This program reads an undefined number of \mbox{\hyperlink{classLanguage}{Language}} objects from the files passed as parameters to \mbox{\hyperlink{main_8cpp_a0ddf1224851353fc92bfbff6f499fa97}{main()}}. It obtains as result the union of all the input \mbox{\hyperlink{classLanguage}{Language}} objects. The result is then sorted by decreasing order of frequency and alphabetical order of bigrams when there is any tie in frequencies. Finally the resulting \mbox{\hyperlink{classLanguage}{Language}} is saved in an output file. The program must have at least an input file and an output file. The output \mbox{\hyperlink{classLanguage}{Language}} will have as language identifier, the one of the first file ($<$file1.\+bgr$>$). If an input file $<$file$\ast$.bgr$>$ has a language identifier different from the one of the first file ($<$file1.\+bgr$>$), then it will not be included in the union. Running example\+: \begin{quote}
language2 $<$file1.\+bgr$>$ \mbox{[}$<$file2.\+bgr$>$ ... $<$filen.\+bgr$>$\mbox{]} $<$output\+File.\+bgr$>$ \end{quote}
